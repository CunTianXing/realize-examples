## Realize Examples

Install Realize:

    $ go get github.com/tockins/realize

From this directory:
 
    $ realize run 

Examples List:

- Print: Print a messagge for 10 times each 2 seconds 
- Server: Starts a web server with "net/http" on 0.0.0.0:5000
- Echo: Starts a web server with "Echo Framework" on 0.0.0.0:3000
